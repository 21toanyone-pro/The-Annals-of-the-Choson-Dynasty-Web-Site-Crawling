# 조선왕조실록 웹 사이트 데이터 크롤링



## 요구 사항

* Python 3.6

* BeautifulSoup
```
pip install beautifulsoup4
```
* selenium
```
pip install selenium
```
* ChromeDriverManager
```
pip install webdriver-manager
```

## 사용 방법

* text crawling

```
python new_crawling.py
```

* Image download

```
python Image_Crawling.py
```
